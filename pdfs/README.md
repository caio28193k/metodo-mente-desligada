## Como exportar para PDF

1. Abra o arquivo HTML no navegador.
2. Pressione `Ctrl + P`.
3. Escolha `Salvar como PDF`.
4. Em mais configurações, ative `Gráficos de segundo plano`.
5. Use papel `A4`, orientação `Retrato` e margens padrão.

Os arquivos já usam `@page { size: A4; margin: 14mm; }` e quebras de página entre cada folha.
